
### 4. Relational Operators (Operadores Relacionales)

Los operadores relacionales comparan dos valores y devuelven un valor booleano (`true` o `false`).

- **Mayor que (`>`)**
- **Menor que (`<`)**
- **Mayor o igual que (`>=`)**
- **Menor o igual que (`<=`)**

**Ejemplo:**

```js
console.log(5 > 3); // Resultado: true
console.log(5 <= 5); // Resultado: true
```
